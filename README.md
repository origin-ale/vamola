# VaMoLA: Variational Montecarlo for Light Atoms
Small project for the 2025/26 Computational Materials Physics course at the University of Bologna.

Based on Thijssen ยง12.2, this project uses Variational Monte Carlo methods to find the ground state energy of:
- the 1D harmonic oscillator
- the hydrogen atom
- the helium atom

The program functions, but the harmonic oscillator and hydrogen are untested; helium is wrong.

## References
- J. M. Thijssen (2007). *Computational physics*, 2nd ed. Cambridge University Press.
